# Lightswitch

Control and monitor hue devices from your terminal.

```bash
$ lightswitch
Manage Hue devices from your terminal

Usage:
  lightswitch [command]

Available Commands:
  completion  generate the autocompletion script for the specified shell
  help        Help about any command
  tui         Device management Terminal UI

Flags:
      --config string   config file (default is $HOME/.lightswitch)
  -h, --help            help for lightswitch

Use "lightswitch [command] --help" for more information about a command.
```
